Python函数——map()和reduce()

map()函数

map()函数接收两个参数，一个是函数，另外一个是迭代器，map的作用就是将传入的函数一次作用到序列的每个元素，并把结果作为新的迭代器返回。

reduce()函数

reduce()函数也接收两个参数，一个是函数，一个是迭代器；这个函数必须接收两个参数，reduce的作用就是把结果和序列的下一个元素做累积计算。
reduce( func, [1, 2,3] ) = func( func(1, 2), 3)
